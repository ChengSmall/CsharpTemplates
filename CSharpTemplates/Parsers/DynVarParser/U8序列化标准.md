# U8序列化标准

使用`Utf-8`编码重新定值部分数据存储规范

## U8格式的改变

### Utf8字符串值

向后读取4字节 1byte = 1-8bit 2byte = 9-16bit ... 依次组成31位整数（忽略最后一位），表示字节数

向后读取指定字节数，以utf-8编码的字符串

### Utf8字符串key

向后读取3字节 第一个字节 1-8bit，第二个字节9-16bit，第三字节的前4bit 17-20bit；

组合为20bit整数，范围在[1,1048575]，作为key的字节数

向后读取指定字节数，以utf-8解码为字符串

### 包含U8格式的字典格式

Loop:

向后读取一个字节，并查看该字节值，如果是0，表示键值对读取到结尾，读取完毕

如果非0，表示存在元素:

继续向后读取key模式的Utf8字符串，视为key

继续向后读取一个值，作为value

继续读取下一个元素 => Loop
